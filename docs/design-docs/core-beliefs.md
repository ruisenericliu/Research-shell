# Core Beliefs

Foundational principles for the Stock Valuation harness.

1.  **Repository as System of Record:** If it's not in the repo, it doesn't exist. All decisions, plans, and technical debt must be captured in Markdown.
2.  **Agent Legibility First:** Code and documentation are optimized for agent navigation and understanding.
3.  **Mechanical Enforcement of Taste:** Use linters, types, and automated checks to enforce architectural boundaries and "golden principles."
4.  **Boring is Better:** Favor stable, composable, and well-documented technologies that are easy for agents to model.
5.  **Continuous Garbage Collection:** Pay down technical debt incrementally. Technical debt is a high-interest loan.
6.  **Progressive Disclosure:** Use `AGENTS.md` as a map, leading agents to deeper sources of truth as needed.
