# Design Documents

Index of architectural and design decisions for the Stock Valuation harness.

- [Core Beliefs](core-beliefs.md): Foundational principles for agent-first development.
- [Code Design Standards](DESIGN.md): Type safety, formatting, exception handling, dependency rules, and layer integrity.
- [Reliability Standards](RELIABILITY.md): Fail-fast principles, 10s timeout, structured JSON logging, and merge bar.
- [Security Standards](SECURITY.md): Secrets in .env, no credentials in logs, external data validation.
- [Frontend Standards](FRONTEND.md): Self-contained HTML, no build tools, CDN-only dependencies.
- [Testing Standards](TESTING.md): 95% coverage target, mocking rules, fixture conventions, and merge bar.
